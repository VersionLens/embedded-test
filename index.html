This page does not inherit any css of the embedded component

As the embedded component is loaded inside of shadowDom

And loaded css is encapsulated and takes an effect, only inside of related shadow dom!

Also, if the root page contains elements with the same class names as embedded component has, it doesn't take any effect

As you can see, the parent element has a different styling for .btn-default, and it does not break "See Details" button in the embedded component
